# inter3semestre
Sobre o Projeto

Este projeto tem como objetivo desenvolver um sistema para melhorar o controle e a organização das operações diárias de uma empresa. A solução proposta concentra-se principalmente no controle de estoque, acompanhamento de documentos de entrada e saída e na gestão centralizada de lançamentos, oferecendo maior integridade dos dados e suporte para a tomada de decisão.

O projeto inclui modelagem, telas em HTML/CSS, scripts SQL com inserts e classes no padrão de modelo (package model).


Funcionalidades Principais

Registro e consulta de produtos

Controle de estoque (entrada e saída)

Gestão de documentos

Cadastro e gerenciamento de usuários

Integração das informações em uma interface simples e padronizada
